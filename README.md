# Splash Wallpaper
![预览](https://raw.githubusercontent.com/berlinxiao/Splash/master/image/shows.jpg)

## 简介
* 一款轻量化的安卓Unsplash客户端
* 来自无版权图片网站Unsplash.com的资源
* 含有超过100万张图片
* 遵循Material Design设计
* 支持查看用户个人资料，策划的收藏，照片统计
* 支持黑暗主题
* 支持原片下载

## 使用方法  
* 前往 https://unsplash.com/developers 创建开发者帐户，申请Unsplash API
* 复制您的访问密钥在 ``RetrofitClient`` 中替换 ``String API_KEY`` 的内容
